## Install VIM config##

- create folders:
```bash
mkdir ~/.vim/backup && mkdir ~/.vim/undos && mkdir ~/.vim/bundle && mkdir ~/.vim/swaps
```

- Install [vundle.vim](https://github.com/VundleVim/Vundle.vim)

- Run :PluginInstall
